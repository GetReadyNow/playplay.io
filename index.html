The Game

It's easy to join. Create a #pong channel and invite @pongbot to it. Each user says "pongbot register" ...

... then challenges someone to a match ...

... the loser says "pongbot lost".

Check your rankings with "pongbot leaderboard" and "pongbot rank".

See detailed usage for other commands.
